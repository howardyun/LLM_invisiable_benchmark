# Zero-Size Font 攻击与检测工具

Zero-Size Font 是一种隐形文本注入攻击技术，通过在 PDF 文档中使用零大小或极小尺寸的字体添加隐藏文本，使其在视觉上不可见，但可以被文本提取工具获取。

## 功能介绍

### 攻击功能 (zero-size-font_attack.py)

创建包含零大小字体文本的 PDF 文件，支持以下攻击类型：

1. **size0**: 完全零大小的字体，在大多数查看器中不可见
2. **size0.01**: 几乎零大小的字体，在高分辨率显示下可能隐约可见
3. **size1**: 非常小的字体，Word 等工具的最小支持尺寸

主要特性：
- 支持在指定页面或所有页面添加隐藏文本
- 支持自定义文本位置或随机位置
- 支持自定义字体、颜色和旋转角度
- 交互式命令行界面

### 检测功能 (zero-size-font_detect.py)

检测 PDF 文件中的零大小字体文本，特性：
- 可配置字体大小阈值（默认 1.0pt）
- 识别并输出所有可疑文本的详细信息
- 显示文本内容、字体、大小和边界框
- 支持扫描整个 PDF 文件

## 使用方法

### 攻击工具使用

#### 交互式使用

```bash
python zero-size-font_attack.py
```

按照提示操作：
1. 输入原始 PDF 文件路径
2. 输入输出 PDF 文件路径（可选，默认在原文件名后加 "_attacked"）
3. 选择攻击类型（1-3）
4. 输入要添加的隐藏文本（多行，空行结束）


### 检测工具使用

#### 交互式使用

```bash
python zero-size-font_detect.py
```

按照提示输入要检测的 PDF 文件路径。



## 技术原理

### 攻击原理

1. **零大小字体**: 将文本的字体大小设置为 0 或极小值
2. **视觉隐形**: 零大小的文本在视觉上不可见，但仍保留在 PDF 的文本流中
3. **文本提取**: 文本提取工具（如 PDF 解析器、复制粘贴功能）仍能获取这些隐藏文本
4. **位置控制**: 可以在文档的任何位置添加隐藏文本

### 检测原理

1. **扫描文本块**: 解析 PDF 文档的文本块和文本行
2. **检查字体大小**: 查找字体大小小于阈值的文本
3. **输出可疑文本**: 记录并显示所有可疑的隐藏文本


## 注意事项

1. **法律合规**: 本工具仅用于教育和研究目的，请勿用于非法活动
2. **检测限制**: 目前的检测工具无法检测完全 size0 的攻击，因为 PyMuPDF 会忽略大小为 0 的文本
3. **查看器差异**: 不同的 PDF 查看器对零大小字体的处理方式可能不同
4. **版本兼容性**: 确保使用最新版本的 PyMuPDF 库

## 检测结果解读

检测结果包含以下信息：

- **text**: 隐藏文本的内容
- **font**: 文本使用的字体
- **size**: 文本的字体大小
- **bbox**: 文本的边界框坐标 (x1, y1, x2, y2)

## 技术限制

1. **PyMuPDF 限制**: 无法检测完全 size0 的文本，因为 PyMuPDF 会忽略大小为 0 的文本
2. **检测阈值**: 需要根据实际情况调整字体大小阈值
3. **复杂文档**: 在复杂布局的 PDF 中可能存在误报
