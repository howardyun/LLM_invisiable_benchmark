# 错位文本PDF生成工具

## 功能介绍

本工具用于生成包含特殊错位文本效果的PDF文件，主要实现以下功能：

1. 创建PDF文档并设置基本属性
2. 对句子中的文本进行特殊排版处理
3. 将句子中的第一个单词与其他单词在视觉上错位显示
4. 生成具有特殊视觉效果的PDF文件

## 文件说明

- `Misaligned_Main.py`: 主程序文件，包含生成错位文本PDF的核心功能
- `README.md`: 本说明文档

## 依赖库

程序依赖以下Python库：

- `reportlab`: 用于创建PDF文件和绘制文本内容
- `pypdf`: 用于从PDF文件中提取文本内容

## 安装依赖

```bash
pip install reportlab pypdf
```

## 功能实现细节

### 1. 文本处理

程序提供了两个辅助函数：

- `extract_text_from_pdf`: 从PDF文件中提取文本内容
- `split_into_sentences`: 将文本粗略地分割成句子

### 2. 创建错位文本PDF

`create_misaligned_pdf` 函数用于创建包含错位文本效果的PDF文件：

- 设置PDF页面大小为标准的letter尺寸
- 设置默认字体为Helvetica，大小为12
- 如果提供了输入PDF路径，则从该PDF中提取文本
- 否则使用预定义的句子列表

### 3. 文本错位处理

程序对每个句子执行以下错位处理：

1. 将句子按空格分割成单词列表
2. 取出句子中的第一个单词
3. 正常绘制整个句子
4. 同时将第一个单词重新绘制在视觉上错位的位置（向右偏移）
5. 形成一种特殊的视觉错位效果

### 4. 坐标计算

- 使用字符间距的近似值（每个字符7个单位）来计算错位位置
- 通过调整Y坐标实现行与行之间的间距
- 支持自动分页功能

## 使用方法

1. 确保已安装所需依赖库
2. 运行脚本：

```bash
python Misaligned_Main.py
```

3. 按照提示输入：
   - 原始PDF文件路径（可选，直接回车使用默认句子）
   - 输出PDF文件路径（可选，直接回车使用默认名称）

4. 运行后将生成一个包含错位文本效果的PDF文件（默认名称：`misaligned_output.pdf`）

## 注意事项

1. 字符间距是通过近似值计算的，可能需要根据实际字体和大小进行调整
2. 可以根据需要修改句子内容和数量
3. 可以调整错位的偏移量来改变视觉效果
4. 默认输出文件名可以通过函数参数进行修改

## 示例效果

生成的PDF文件中，每个句子的第一个单词会同时出现在两个位置：
1. 正常的句子流中
2. 向右下方偏移的错位位置

这种效果会在视觉上产生一种干扰或特殊的排版风格。
