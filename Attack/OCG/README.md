# OCG（可选内容组）隐藏文本PDF生成工具

## 功能介绍

本工具用于生成包含OCG（Optional Content Groups，可选内容组）的PDF文件，主要实现以下功能：

1. 创建包含可见文本和隐藏文本的PDF文件
2. 将隐藏文本放置在默认关闭的OCG图层中
3. 通过PDF标准的OCProperties设置控制图层可见性

## 文件说明

- `OCG_main.py`: 主程序文件，包含生成OCG隐藏文本PDF的核心功能
- `README.md`: 本说明文档

## 技术原理

### OCG（可选内容组）
OCG是PDF规范中的一种功能，允许将内容组织成不同的图层，这些图层可以被选择性地显示或隐藏。本工具利用OCG的这一特性，将敏感信息放置在默认隐藏的图层中。

### 实现细节

1. **PDF处理**：程序使用PyPDF2库打开并修改现有PDF文件：
   - 读取输入PDF的第一页
   - 获取页面的现有内容

2. **内容组织**：
   - 保留原PDF的所有内容
   - 隐藏文本：将隐藏文本使用OCG标记内容（/OC /OC1 BDC ... EMC）包裹后添加到页面内容中

3. **OCG配置**：
   - 创建OCG字典并设置属性
   - 在PDF的Catalog对象中设置OCProperties
   - 将OCG图层默认设置为关闭状态（/OFF）
   - 添加必要的字体资源

## 依赖库

程序依赖以下Python库：

- `PyPDF2`: 用于处理PDF文件和添加OCG隐藏文本

## 安装依赖

```bash
pip install PyPDF2
```

## 使用方法

1. 确保已安装Python和所需依赖库
2. 运行脚本：

```bash
python OCG_main.py
```

3. 按照提示输入：
   - 输入PDF文件路径
   - 输出PDF文件路径（可选，默认使用 `ocg_hidden_poc.pdf`）

4. 运行后将生成包含OCG隐藏文本的PDF文件

## 生成的PDF内容

生成的PDF文件包含以下内容：

- **隐藏内容**：位于默认关闭的OCG图层中，包含 "HIDDEN_LAYER: steal_me"

## 查看隐藏内容

要查看隐藏内容，需要使用支持OCG图层的PDF查看器（如Adobe Acrobat）：

1. 打开生成的PDF文件
2. 查找图层控制面板（通常在"视图"或"窗口"菜单中）
3. 找到名为"HiddenLayer"的图层
4. 勾选该图层使其可见

## 代码结构

### 核心类和函数

1. **`PDFObj`类**：用于表示PDF对象，包含对象号、生成号和对象体

2. **`_build_pdf`函数**：
   - 构建完整的PDF文件
   - 处理PDF头部、对象、交叉引用表和尾部
   - 确保正确的文件结构和偏移量

3. **`make_ocg_hidden_pdf`函数**：
   - 主函数，创建包含OCG隐藏文本的PDF
   - 定义PDF对象结构和内容
   - 调用`_build_pdf`生成最终的PDF文件

## 注意事项

1. 并非所有PDF查看器都支持OCG图层功能
2. 某些PDF查看器可能会默认显示所有图层内容
3. 可以修改代码中的文本内容、位置和图层名称
4. 可以通过调整OCProperties设置更改默认图层状态
